<h2>Инструкция к запуску</h2>

После клонирования репозитория в корне приложения прописываем:
<ol>
  <li>npm i</li>
  <li>npm run start</li>
</ol>

Примечание:
<ul>
  <li>Данные лежат: ./src/state.json</li>
  <li>Инструменты:
    <ul>
      <li>React</li>
      <li>Styled Component</li>
      <li>React Tabs</li>
    </ul>
  </li>
</ul>
